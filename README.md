# Main Repository Containing Odin
Odi: Sequence Analysis Pipeline primarily for Asystbio Laboratories


The two main packages are contained in their respective folders.  The Valkyries folder contains libraries in common to both.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

**Do Not Use setup.py To Install**
